# Saathvik Portfolio (React + HTML + CSS)

Interactive portfolio built with React routing, project filters, and dedicated project detail pages.

## Run locally

1. Install Node.js LTS (includes npm)
2. Install dependencies
3. Start development server

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```

## Replace placeholders

- Edit `src/data/projects.js` with your real project details and links.
- Edit personal info in `src/components/Hero.jsx` and `src/pages/Home.jsx`.
